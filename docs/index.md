# About
These docs contain information on how to build, adapt, and deploy my website.

# Technologies
This website was built using Gridsome, a Vue-based, Gatsby.js-like framework
that uses the JAMStack and prerenders data before deployment.

# Deployment
This website is set up to be deployed on Firebase Hosting.

Run `npm run lint` to lint the code before deployment.

If Firebase has been initialized using `firebase init`, run `npm run build` and
then `firebase deploy` to build and deploy the website to Firebase Hosting.
